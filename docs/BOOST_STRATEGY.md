## 1. Контент-маркетинг (самый мощный долгосрок)

### Блог / статьи

Каждая статья — шанс занять топ по новому запросу.

| Запрос | Идея статьи |
|--------|-------------|
| «натальная карта как читать» | Гайд для начинающих |
| «соляр на год что это» | Объяснение + CTA на услугу |
| «совместимость по дате рождения» | Как считается, примеры |
| «астролог онлайн отзывы» | Кейсы (с согласия клиентов) |
| «когда нужен астролог» | 5–7 ситуаций + запись |
| «астрология отношения» | Связь с relationshipastrology |

**Формат:** 1500–3000 слов, H2/H3, ключевые фразы, ссылка на консультацию.  
**Результат:** Органический трафик через 2–4 месяца, долгосрочный рост.

### Доп. лендинги под города

Отдельные страницы без офиса, но с гео-запросами:

- `/astrolog-online` — «астролог онлайн»
- `/astrolog-moscow` — «астролог москва» (если работаете с РФ)
- `/astrolog-russia` — «астролог россия»

Текст: «Работаю онлайн с клиентами из Москвы, Алматы, России, Казахстана…»

## 2. Технические ускорители индексации

### IndexNow

Мгновенный пинг при обновлении страниц (Яндекс, Bing).

- Сервис: https://www.indexnow.org/
- Или плагин/скрипт при деплое

### Google Indexing API

Ручной запрос индексации ключевых URL (через Search Console).

### RSS / sitemap для подписок

RSS-лента блога — быстрее краулятся новыми страницами.

## 3. Конверсия и доверие

### Отзывы

- Блок «Отзывы» на главной (3–5 шт.)
- Schema `Review` / `AggregateRating` — звёзды в выдаче
- Ссылка на WhatsApp после каждого отзыва

### Social proof

- «N клиентов за 2024»
- «Консультации на русском и английском»
- Логотипы/цитаты, если есть партнёры

### Усиление CTA

- Повтор кнопки «Записаться» после каждого блока
- Exit-intent: всплывающее окно «Остались вопросы? WhatsApp»
- Sticky-кнопка WhatsApp в углу на мобильных

## 4. Соцсети как источник трафика

### Instagram / Telegram

- В каждом посте — «подробнее на сайте» / «запись в био»
- Карусели: «5 причин обратиться к астрологу» → ссылка на статью
- Сторис с вопросом → «ответ в блоге на сайте»

### Взаимопиар

- Гостевые посты у психологов, коучей
- Упоминания в подкастах про саморазвитие
- Коллаборации с инфлюенсерами в нише
