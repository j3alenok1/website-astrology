# Цели Яндекс.Метрики для ASTRO by NDAUZH

Документ описывает все цели, которые отправляются в Яндекс.Метрику, и инструкции по их настройке.

## Как создать цели в Метрике

1. Откройте [Яндекс.Метрику](https://metrika.yandex.ru) → счётчик **106988269**
2. **Настройки** → **Цели** → **Добавить цель**
3. Тип цели: **JavaScript-событие**
4. Идентификатор цели: укажите `Идентификатор` из таблицы ниже

---

## Рекомендации по анализу

1. **methodology_detail** — сравните частоту по `service`: какая методология (Астрология, Таро, Психология, Комплексный) привлекает больше внимания.
2. **click_product** — по параметру `product` видно, какие продукты чаще выбирают.
3. **faq_open** — по `index` видно, какие вопросы раскрывают чаще (0 = первый, 1 = второй и т.д.).
4. **scroll_depth** — воронка вовлечённости: сколько доходят до 50%, 75%, 100%.
5. **click_payment_intent** vs **lead_submit** — сколько хотят оплатить сразу vs оставить заявку.
